# Phase 4.3 — Medical necessity reasoner + RAG over NCDs/LCDs

**Week**: 4 · **Status**: 🔄 In progress

## Goal

Add the reasoning node: given the evidence package from 4.2 and a target service, produce a structured medical-necessity argument grounded in the payer's published policy.

## Scope — what ships

1. **Policy corpus ingestion**
   - **Tier 1**: CMS NCDs (National Coverage Determinations) — clean, structured, public.
   - **Tier 2** (stretch): 2–3 public payer LCDs (Aetna / UHC / Humana) for MRI lumbar spine.
   - Each policy chunked at the paragraph level with `policyId + section + paragraph` metadata preserved.
   - Embedding model choice gated by ADR-0004; vector store choice gated by ADR-0005.

2. **Medical Necessity Reasoner node**
   - Input: evidence package (from 4.2) + target service + payer ID.
   - Process: retrieve top-k policy paragraphs → for each policy criterion, decide `met` / `not_met` / `unclear` with the evidence resource(s) backing it.
   - Output: structured `Decision` object — `[(criterion, status, evidence_refs, citation)]`.

3. **Deterministic citation checker**
   - Every citation emitted by the Reasoner is verified to exist in the corpus (paragraph hash match). Hallucinated citations are rejected at the node boundary, not at the UI.

4. **Demo**
   - `make demo-reason PATIENT=<id> SERVICE=mri-lumbar-spine PAYER=cms-medicare`
   - Output: human-readable necessity argument + structured Decision JSON.

## ADRs likely needed

- **ADR-0004** — Embedding model choice (e.g., `text-embedding-3-large` vs an open-weights medical embedding model).
- **ADR-0005** — Vector store (pgvector vs LanceDB vs Qdrant) for the policy corpus.
- **ADR-0006** — Model abstraction layer — interface between Reasoner and underlying LLM provider, so evals can run against ≥2 model families.

## Eval gates

| Metric | Target |
|---|---|
| Citation grounding rate (deterministic check) | 1.00 |
| Criterion-level accuracy vs clinician-labeled golden set (n=30) | ≥ 0.85 |
| Hallucination rate on "no-evidence" cases (must say "unclear", not invent) | < 0.05 |
| End-to-end latency (P50) | < 20s |
| Cost per Reasoner run | < $0.10 |
